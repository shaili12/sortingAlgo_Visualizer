# sortingAlgo_Visualizer

<b>A JavaScript web application for visualizing various sorting algorithms. Built for use by developers and educators alike. Implemented algorithms include:</b>

<b>Bubble Sort</b>
<b>Selection Sort</b>
<b>Insertion Sort</b>
<b>Merge Sort</b>
<b>Heap Sort</b>
<b>Quick Sort</b>
